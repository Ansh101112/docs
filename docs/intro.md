# Introduction

Welcome to **Jabrod** - the RAG as a Service platform. Build AI-powered applications with knowledge bases in minutes.

## What is Jabrod?

Jabrod is a cloud platform that lets you:

- **Create Knowledge Bases** - Upload documents (PDFs, text files, etc.)
- **Semantic Search** - Query your documents with natural language
- **AI Chat** - Get AI-generated responses based on your content
- **Simple SDK** - Integrate in minutes with our TypeScript SDK

## Use Cases

| Use Case | Description |
|----------|-------------|
| Customer Support | Build AI chatbots that answer questions from your support docs |
| Documentation Search | Help users find relevant information across your docs |
| Internal Knowledge | Create an AI assistant for your company's internal knowledge |
| Content Analysis | Extract insights from large document collections |
